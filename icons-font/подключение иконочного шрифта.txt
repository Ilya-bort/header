ИКОНОЧНЫЙ ШРИФТ:

каталог иконок:
https://fontawesome.com/kits/cf457b3d7a/use?welcome=yes

заходим во вкладку icons и копируем классы иконок
пример:
<i class="fa-brands fa-youtube"></i>


ГЛОБАЛЬНОЕ ПОДКЛЮЧЕНИЕ:

скрипт для глобального подключения: 
<script src="https://kit.fontawesome.com/cf457b3d7a.js" crossorigin="anonymous"></script>
+ подключаем классы в html


ЛОКАЛЬНОЕ ПОДЛЮЧЕНИЕ ИКОНОК С FONT AWESOME В ПРОЕКТ:

*скачиваем архив с сайта
*подключаем файл all.css в папку css проекта, а если нужно что-то конкретное, то подключаем файлы fontawesome.css и файл конкретных иконок (например brands.css) в папку css проекта
*подключаем папку  webfonts в корень проекта
*подключаем all.css в html и переназначаем стили именно для before, а не для всего блока
*подключаем классы в html


СОБСТВЕННЫЙ ИКОНОЧНЫЙ ШРИФТ:

сайт для создания собственного шрифта:
https://icomoon.io/app/#/select

*наполняем папку icons-font в корне проекта иконками
*загружаем иконки на сайт, корректируем и генерируем шрифт
*килаем архив в папку icons-font и распаковываем
*берем папку fonts и файл стилей style.css, который надо переименовать
*кидаем в корень проекта папку fonts, а файл стилей в папку css
*подключаем файл стилей в html и меняем путь к шриту в этом файле (приписываем ../)
*подключаем нужные классы в html
